网络资源收集

发表于 2016-03-14   |   次阅读

这篇文章旨在收集个人觉得比较赞的一些网络资源，留作以后参考。这样可以免去重复搜索的麻烦，亦可以免去收藏夹的混乱，用一片文章列出来，一目了然。而且，走哪儿都可以访问。还可以自由的添加备注信息，还能不断地补充和修改。

  1. Android 开发
    官方API，你懂的
    http://developer.android.com/develop/index.html
    极客学院Android学习线路图
    http://www.jikexueyuan.com/path/android/
    开发技术前线 高质量技术文章的聚合网站
    http://www.devtf.cn/
    ADB调试，入门
    需要添加一个连接命令：adb connect [device IP]
    http://www.cnblogs.com/meil/archive/2012/05/24/2516055.html
    Android 开发中文站
    http://www.androidchina.net/
  2. Github & Git
  3. Hexo
    Hexo官方链接，玩儿Hexo绝对值得收藏
    https://hexo.io/
    岁月如歌的博客：一篇关于Hexo搭建教程，还有些步骤（比如评论功能）没有完成，以后有需求要再次参考

    Markdown入门指南，对于现在还不知道Markdown是什么东东的童鞋，有时间要看一下
    http://www.jianshu.com/p/1e402922ee32/#
  5. 数据库
    SQLite Manager官网，图形化数据库查看&管理软件
    http://www.sqlabs.com/sqlitemanager.php
  6. JAVA
    Google Java编程风格指南
    http://www.hawstein.com/posts/google-java-style.html
